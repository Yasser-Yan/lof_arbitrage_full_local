# LOF套利监控

这是一个本地运行的 LOF 套利观察工具，用来查看场内 LOF 的现价、现价日期、基金净值、净值日期、溢价率、申购状态和赎回状态。

## 文档入口

- 普通用户请看：[用户使用说明](./USER_GUIDE.md)
- 开发和维护请看：[开发使用说明](./DEVELOPER_GUIDE.md)

## 发布文件

最新可交付文件在 `发布` 文件夹：

```text
发布\LOF套利监控.exe
发布\LOF套利监控安装包.exe
发布\funds_config.json
发布\配置文件说明.txt
```

推荐普通用户使用 `LOF套利监控安装包.exe`。如果只是临时运行，也可以把 `LOF套利监控.exe` 和 `funds_config.json` 放在同一目录后双击运行。

## 快速启动开发版

已安装 Node.js 时，双击：

```text
START.bat
```

或在项目目录运行：

```text
node server.js
```

启动后访问：

```text
http://127.0.0.1:8787
```
